📋 輸入 Firebase 設定
📖 設定教學（約 5 分鐘）
1
打開 console.firebase.google.com（需要 Google 帳號登入）
2
按「新增專案」→ 填入名稱（例：NZ Trip 2026）→ 停用 Google Analytics → 建立專案
3
左側選單按「Firestore Database」→「建立資料庫」→ 選「以測試模式開始」→ 選任意位置 → 啟用
4
返回 Firebase 主頁 → 按「</> 新增應用程式」（網頁圖示）→ 填入名稱 → 繼續
5
複製 firebaseConfig = { ... } 大括號內嘅 JSON 內容
6
貼入上方輸入框，設定旅行密碼，按「儲存並開始同步」
7
其他手機重複步驟 6，用同一個旅行密碼即可共享所有資料 ✅

🗺️ Google Maps 設定

輸入 Google Maps API Key 後，地點頁面會顯示互動地圖，並可計算兩點之間的駕車時間。
📖 取得 API Key 步驟
1
打開 console.cloud.google.com，選擇你的 Firebase 項目（travel-plan-627c0）
2
左側選單 → API 和服務 → 程式庫，搜尋並啟用：
① Maps JavaScript API
② Directions API
3
左側 → 憑證 → 建立憑證 → API 金鑰，複製金鑰
4
（建議）點擊金鑰 → API 限制，限制只允許上述兩個 API；應用程式限制設為你的 GitHub Pages 網址
